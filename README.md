#### 应用简介
纸飞机是一款基于**MVP**实现的**Material Design**风格的简单阅读应用，数据主要来源于知乎、豆瓣、果壳的开源API，实现了**夜间模式**，**无图浏览**，**数据缓存**，**个人收藏**，**文章图片本地保存**等功能。
- 网络获取主要使用了Volley

- 使用SQLite缓存数据

- 自定义commonRecyclerAdapter实现单一布局和简单多布局的支持，上拉加载更多

- 自定义内置浏览器，页面链接可选内置浏览器或自带浏览器打开

- 选择日期浏览当天的新闻

- 随机获取单一新闻阅读

#### 主要使用的库
- volley
- gilde
- customtabs
- cardview
- gson
